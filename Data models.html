Data Models:

  • Relational model : The most common model, where data is organized into tables with rows and columns, with relationships defined through foreign keys.
  • Entity-Relationship model(ERM) : Represents entities(people,places,things) and their relationship using visual diagrams.
  • Object-Oriented Model : Extends the relational model by incorporating object-Oriented concepts like inheritance and encapsulation.
  • Hierarchical Model : Data is structured in a tree-like Hierarchy with parent-child relationships.
  • Network Model : Allows complex relationships between data items through a network structure.
