Database Users:

  • Database Administrator(DBA) :

    • Responsible for managing the database, including security,performance, backup, and recovery.

  • Application Developers:

    • Create applications that interact with the database to retrive and update data.

  • End Users:

    • Individuals who use application built on the database to access and view information

  • Data Analysts:

    • Extract and analyze data from the database to generate insights.
